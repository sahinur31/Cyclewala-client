    <ol>

                <li>Home Page with a footer section</li>
            </ul>
        </li>
        <li> Log in Page
            <ul>
                <li>If a user Register with email password, he or she log in with this page or use Google sign in, then user log in  </li>

        <li> Register in Page
            <ul>

            <ul>
